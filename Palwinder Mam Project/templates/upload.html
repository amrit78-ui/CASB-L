Batch Prediction via CSV / Excel

Upload a student records CSV or Excel spreadsheet to generate predictions, rank profiles by reward score, and export comprehensive reports.

How It Works

Step 1
Upload Your File

Drag & drop or browse a .csv, .xlsx or .xls student dataset file.

Step 2
Click Submit

Preview the selected file, then press Submit to start the IRL inference engine.

Step 3
View Top 30 Rankings

See ranked student profiles sorted by IRL Policy Reward Score with learning & behavior predictions.

Step 4
Export Excel Report

Download a complete Excel report of all students evaluated, sorted by reward score.

What This System Predicts
Learning Speed Category

Classifies each student as Fast Learner, Average Learner, or Slow Learner based on their academic and lifestyle patterns.

Behavior Category

Identifies students as Self-Motivated, Dependent Learner, or At-Risk Student using behavioral signals.

IRL Reward Score

A policy reward score derived from Inverse Reinforcement Learning — higher = stronger academic behavior. Used to rank all students.

Expected File Format

Your spreadsheet should contain student data with these key columns. Missing columns are handled automatically with defaults.

Name of the Student
Program / Course
Semester
Gender, Age, Nationality
Study Time, Sleep Hours, Stress Level
Attendance %, Social Media Hours
Class Participation, Note-taking Habits
... + 50 additional feature columns
Column headers are matched automatically even if slightly different.
Output Columns in Excel Report

After processing, the downloaded Excel report includes the following fields for all evaluated students:

Student Name
Program / Course
Semester
Learning Speed Category
Behavior Category
Behavior Sentiment +ve / -ve
Reward Score (sorted highest → lowest)
